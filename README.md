# Voting App — صفحة هبوط (Landing Page)

هذه الصفحة عبارة عن صفحة هبوط لتطبيق "Voting App".

**الملفات الرئيسية:**

- `index.html` — الصفحة الرئيسية (HTML بالعربية RTL).
- `assets/` — تحتوي على `styles.css` والصور (screenshots) و`cover.svg`.
- `Voting-app-UPDATE-v5.1.0.apk` — ملف APK الحديث (v5.1.0).

**النشر على GitHub Pages:**

1. رفع الملفات إلى المستودع على `main` أو فرع آخر.
2. تمكين GitHub Pages في الإعدادات واختيار المصدر المناسب.
3. يمكن أيضاً رفع الـ APK ضمن GitHub Releases وربط الصفحة به.
